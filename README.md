# 👋 About Me

Hi!

I'm **Mohamed Tarek**, an Embedded Software Engineer passionate about low-level programming, embedded systems architecture, communication protocols, and automotive software development.

This repository documents my journey of building reusable embedded drivers and complete embedded applications from scratch using pure Embedded C.

The primary objective of this portfolio is not only making projects work, but designing them using modular, scalable and maintainable software architecture similar to industrial embedded systems.
