# DC/Servo Motor Control

## 📖 Description

This project demonstrates DC motor and Servo motor control using the ATmega32 microcontroller. The motor can be started, stopped, and controlled through software while utilizing a reusable motor driver abstraction layer.

---

## ✨ Features

- Motor start and stop control.
- Direction control.
- Angle Setting 
- Modular motor driver.
- Easy integration into larger systems.

---

## 🛠 Hardware

- ATmega32
- DC Motor
- Servo Motor
- H-Bridge Driver
