# Frequency and Duty Cycle Detector

## 📖 Description

This project measures the frequency and duty cycle of an external digital signal using the Input Capture Unit (ICU) of Timer1 in the ATmega32 microcontroller.

---

## ✨ Features

- Frequency measurement.
- Duty cycle calculation.
- High accuracy using Input Capture.
- Real-time signal analysis.

---

## 🛠 Hardware

- ATmega32
- Signal Generator
- LCD
