# Digital Clock using Timer0

## 📖 Description

This project implements a digital clock using Timer0 of the ATmega32 microcontroller. The timer generates periodic interrupts to accurately count time and display hours, minutes, and seconds with ability to set specific time to start from.

---

## ✨ Features

- Real-time clock functionality.
- Accurate software timing.
- Hours, minutes, and seconds display.
- Timer0 interrupt utilization.
- Ablity to set the time using Keypad

---

## 🛠 Hardware

- ATmega32
- 7-Segment Display
- Keypad
