# Real-Time Clock

## 📖 Description

This project implements a real-time clock using Timer2 of the ATmega32 microcontroller and 32.768 KHz Crystal Oscillator for real timing . The system continuously keeps track of time while providing accurate clock functionality through timer interrupts with the ability to set the clock with specific time using Keypad.

---

## ✨ Features

- Real-time clock.
- Accurate time keeping.
- Hours, minutes, and seconds display.
- Ability to set the clock with specific time
- Timer2 interrupt utilization.
