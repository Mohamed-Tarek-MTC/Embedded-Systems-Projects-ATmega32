# I2C Communication Protocol

## 📖 Description

This project demonstrates I2C communication between two ATmega32 microcontrollers configured as Master and Slave. It implements a reusable I2C (TWI) driver supporting addressing, acknowledgments, and reliable data exchange.

---

## ✨ Features

- Master/Slave communication.
- Address-based communication.
- Configurable clock frequency.
- Modular I2C driver.

---

## 🛠 Hardware

- 2 × ATmega32
- LCDs
- Keypad
- SDA & SCL Pull-up Resistors

---

## 📚 Drivers Used

### MCAL

- DIO
- I2C (TWI)

### HAL

- LCD
- Keypad

---

## 🧠 Embedded Concepts

- I2C Protocol
- Open Drain Bus
- Master/Slave Architecture
- Addressing
- ACK/NACK Mechanism
- Multi-device Communication
- Driver Abstraction
- Modular Programming
