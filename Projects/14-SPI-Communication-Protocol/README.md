# SPI Communication Protocol

## 📖 Description

This project demonstrates SPI communication between two ATmega32 microcontrollers configured as Master and Slave. The Master exchanges data read through the Keypad with the Slave through the SPI bus, while the received information is displayed on an LCD, demonstrating high-speed synchronous full-duplex communication using a reusable SPI driver.

---

## ✨ Features

- SPI Master configuration.
- SPI Slave configuration.
- Full duplex communication.
- Modular SPI driver.

---

## 🛠 Hardware

- 2 × ATmega32
- Keypad
- LCD
- SPI Bus (MOSI, MISO, SCK, SS)
