# Electronic Voting Machine

## 📖 Description

This project implements a simple electronic voting machine using the ATmega32 microcontroller. Users can cast votes through push buttons, and the system records and displays voting results on the LCD.

---

## ✨ Features

- Multiple candidate support.
- Vote counting.
- Result display.
- Simple user interaction.
- Reliable button handling.

---

## 🛠 Hardware

- ATmega32
- Push Buttons
- LCD
- Resistance

---

## 📚 Drivers Used

### MCAL

- DIO

### HAL

- LCD
- Button
