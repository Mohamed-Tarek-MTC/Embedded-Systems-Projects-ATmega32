# Ultrasonic Distance Measurement

## 📖 Description

This project measures the distance to nearby objects using an HC-SR04 ultrasonic sensor interfaced with the ATmega32 microcontroller. The measured distance is calculated by the ICU using the echo pulse duration and displayed on an LCD.

---

## ✨ Features

- Accurate distance measurement.
- LCD display.
- Trigger and Echo signal processing.
- Reusable ultrasonic driver.

---

## 🛠 Hardware

- ATmega32
- HC-SR04 Ultrasonic Sensor
- LCD
