# Temperature Sensor

## 📖 Description

This project reads temperature values from an analog temperature sensor using the ATmega32 ADC and displays the measured temperature on an LCD.

---

## ✨ Features

- Temperature measurement.
- ADC conversion.
- LCD display.
- Continuous monitoring.

---

## 🛠 Hardware

- ATmega32
- LM35 Temperature Sensor
- LCD
- Battary
- Capacitor

---

## 📚 Drivers Used

### MCAL

- ADC
- DIO

### HAL

- LCD

## 🧠 Embedded Concepts

- ADC

- Voltage-to-Temperature Conversion
