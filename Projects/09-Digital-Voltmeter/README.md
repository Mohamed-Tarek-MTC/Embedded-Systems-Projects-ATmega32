# Digital Voltmeter

## 📖 Description

This project implements a digital voltmeter using the ATmega32 Analog-to-Digital Converter (ADC). Analog voltages are sampled, converted into digital values, and displayed on LCD.

---

## ✨ Features

- Analog voltage measurement.
- ADC conversion.
- LCD display.
- Configurable reference voltage.

---

## 🛠 Hardware

- ATmega32
- Potentiometer
- LCD
- Batteries
- Capacitors

---

## 📚 Drivers Used

### MCAL

- ADC
- DIO

### HAL

- LCD
