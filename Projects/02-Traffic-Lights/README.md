# Traffic Lights

## 📖 Description

This project simulates a real-world traffic light control system using the ATmega32 microcontroller. The traffic lights automatically cycle through Red, Yellow, and Green states with predefined timing intervals displayed on Seven Segments to demonstrate basic traffic management logic on 2 roads.

---

## ✨ Features

- Automatic traffic light sequencing.
- Configurable timing intervals.
- Modular LED driver implementation.
- Modular Seven Segment driver implementation.
- Continuous operation.
