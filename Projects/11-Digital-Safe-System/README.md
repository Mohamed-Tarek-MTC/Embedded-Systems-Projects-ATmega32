# Digital Safe System

## 📖 Description

This project implements a password-protected digital safe using the ATmega32 microcontroller. Users enter a password through a keypad, and the system grants or denies access while displaying status messages on an LCD with ability to reset password after opening the safe.

---

## ✨ Features

- Password authentication.
- LCD user interface.
- Keypad input.
- Access control.
- Invalid password handling.

---

## 🛠 Hardware

- ATmega32
- LCD
- Keypad
- LEDs/Buzzer
