# UART Communication Protocol

## 📖 Description

This project demonstrates UART communication between two ATmega32 microcontrollers. One MCU acts as a transmitter, Sends a string then allows the user to enter data through a keypad, while the second MCU receives the transmitted data and displays it on an LCD. The project showcases reliable asynchronous serial communication using a reusable UART driver.

---

## ✨ Features

- UART initialization.
- Serial data transmission/.
- Serial data reception.
- Configurable baud rate.
- Communication between two ATmega32 MCUs.
- Modular UART driver.
