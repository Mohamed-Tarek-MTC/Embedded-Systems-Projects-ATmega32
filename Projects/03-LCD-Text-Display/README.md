# LCD Text Display

## 📖 Description

This project demonstrates interfacing a 16x2 character LCD with the ATmega32 microcontroller. 
It displays text on both LCD lines while implementing smooth circular scrolling on the first line using a reusable LCD driver.

---

## ✨ Features

- Displays text on a 16x2 LCD.
- Circular scrolling on the first row.
- Static text display on the second row.
- Modular LCD driver implementation.
- Easy integration into larger embedded projects.
